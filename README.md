# 51_task

基于 STC89C52RC / 普中 51 实验板的单片机课程设计，使用 8051 汇编实现数码管显示、矩阵键盘扫描、串口自发自收和电话号码滚屏显示。

## 功能说明

- 上电默认显示 `C875`。
- 4x4 矩阵键盘使用线反转法扫描，按键映射为：`S1->0`，`S2->1`，...，`S10->9`，`S11->A`，...，`S16->F`。
- `K3 / P3.2 / INT0`：切换普通键盘显示模式和串口自发自收模式。
- `K4 / P3.3 / INT1`：切换普通键盘显示模式和电话号码滚屏模式。
- 串口自发自收模式需要短接 `P3.1(TXD)` 与 `P3.0(RXD)`。
- 电话号码模式会通过串口发送一次占位号码 `13800138000`，并在 4 位数码管上循环滚屏显示。

## 硬件连接

- 数码管段码：`P0`
- 数码管位选：`P2.2 ~ P2.4`
- 矩阵键盘：`P1.0 ~ P1.7`
- 串口接收：`P3.0 / RXD`
- 串口发送：`P3.1 / TXD`
- 任务三切换按键：`K3 / P3.2 / INT0`
- 任务四切换按键：`K4 / P3.3 / INT1`

## 文件结构

- `main.asm`：主程序入口、模式分发和模块 include。
- `display.asm`：数码管动态扫描显示、按键值显示、滚屏缓冲显示。
- `keyboard.asm`：矩阵键盘线反转法扫描。
- `interrupt.asm`：外部中断服务程序。
- `uart.asm`：串口初始化、发送和接收。
- `phone.asm`：电话号码发送和滚屏逻辑。
- `delay.asm`：延时子程序。

## 使用方法

1. 使用 Keil 打开 `51_task.uvproj`。
2. 编译工程并生成 HEX 文件。
3. 烧录到普中 51 实验板。
4. 普通模式下按矩阵键盘，数码管显示对应键值。
5. 按 `K3` 进入串口自发自收模式；短接 `P3.1` 和 `P3.0` 后，按矩阵键盘显示回环收到的键值。
6. 按 `K4` 进入电话号码模式，数码管滚屏显示电话号码，串口发送一次号码。

## 串口参数

- 波特率：`9600`
- 数据位：`8`
- 停止位：`1`
- 校验位：无
- 流控：无

如果电脑串口助手显示乱码，请优先检查波特率、晶振频率和串口助手显示模式。任务三发送的是原始键值 `00H~0FH`，按文本显示时可能不是可见字符；任务四发送的是 ASCII 电话号码。

## 修改电话号码

在 `phone.asm` 末尾修改 `PHONE_TABLE` 即可，例如：

```asm
PHONE_TABLE:
    DB '1','3','8','0','0','1','3','8','0','0','0'
```
